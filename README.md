# Premium 3D AI Engineer Portfolio

A bespoke, premium digital portfolio designed for an AI Engineer & Full Stack Developer. Built with React, Three.js (React Three Fiber), Framer Motion, and Vite.

Features a custom 3D hybrid AI visualization inspired by modern LLM architectures, complete with animated attention connections, token streams, and a vector embedding cloud.

## Features
- **3D AI Visualization**: Custom procedural neural network and embedding cloud using Three.js
- **Cinematic Animations**: Scroll-triggered GSAP transitions and Framer Motion layouts
- **Dark Luxury Design**: Premium glassmorphism UI with accent glows
- **Responsive**: Fully functional on mobile with performance-scaled 3D graphics
- **SEO Optimized**: Pre-configured meta tags, sitemap, and robots.txt

## How to Run Locally

To preview the portfolio on your local machine, follow these steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 18 or higher) installed on your system.

### 1. Install Dependencies
Open a terminal in the project directory and run:

```bash
npm install
```

### 2. Start the Development Server
Once the dependencies are installed, start the local Vite server:

```bash
npm run dev
```

### 3. View the Site
The terminal will display a local URL (usually `http://localhost:5173`). Click the link or open it in your browser to view the site.

## Building for Production

To create an optimized production build, run:

```bash
npm run build
```

This will output the compiled assets into the `dist` directory, which is ready to be deployed to Vercel, Netlify, or any static hosting service.
